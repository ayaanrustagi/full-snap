# Full Snap — Capture · Redact · Ship

Chrome extension for **safe, shareable screenshots**: capture, auto-redact secrets, annotate, ship a pack to Slack/Linear. Local-first.

- **Site:** https://ayaanrustagi.github.io/full-snap/  
- **Privacy:** https://ayaanrustagi.github.io/full-snap/privacy.html  
- **Store listing copy:** [STORE_LISTING.md](./STORE_LISTING.md)  
- **Version:** 2.0.1  

## Load unpacked
`chrome://extensions` → Developer mode → **Load unpacked** → this folder.

## Store zip
From `extensions/`: `./package-store.sh` → `full-snap-store.zip`

Privacy Policy

Effective date: July 19, 2026 · Full Snap Chrome extension & this website

Summary: Full Snap works on your computer. We do not operate a Full Snap backend that receives your screenshots, page content, or annotations. There is no Full Snap account.

Who we are

Full Snap is a browser extension and static marketing site. Contact: use the GitHub repository linked from the project homepage or the email listed on the Chrome Web Store listing when published.

Data the extension processes

  • Active tab content (transient): When you start a capture, the extension may scroll the page, read layout metrics, apply temporary on-page overlays for redaction or selection, and use Chrome’s captureVisibleTab API to create image data.
  • Capture images: Held briefly in chrome.storage.session so the local editor tab can stitch and display them, then cleared. Exports (PNG/JPG) and clipboard copies are written only to your device or OS clipboard at your action.
  • Preferences: Options such as capture delay, redaction on/off, hide sticky headers, and stamp on/off are stored in chrome.storage.local.
  • Local history metadata: Optional short list of recent capture times, page titles/URLs, and mode — stored locally only, not for advertising.

What we do not do

  • No Full Snap cloud sync of screenshots
  • No sale of personal data
  • No advertising trackers inside the extension
  • No remote code execution

Permissions

  • activeTab — access the tab you intentionally invoke the extension on
  • scripting — inject capture helpers (scroll, region/element pickers, temporary redaction covers)
  • storage — preferences and short-lived capture handoff to the editor

Redaction disclaimer

Auto-redaction uses best-effort pattern matching in the page DOM (e.g. email-like text, password fields). It can miss content drawn in canvas, images, or unusual formats. Always review before sharing.

Website

This static site may be hosted on GitHub Pages. Hosting providers may log standard request data (IP, user agent) per their policies. The site does not set Full Snap accounts or receive extension capture data.

Children

Full Snap is not directed at children under 13.

Changes

We may update this policy; the effective date above will change. Continued use after updates means you accept the revised policy.

Chrome Web Store

When distributed via the Chrome Web Store, Google’s terms also apply to distribution. This policy describes the product’s own data practices.
